# 1. Descripción del proyecto

Las organizaciones que apoyan a migrantes, como Casa Monarca, manejan información altamente sensible, incluyendo datos personales y documentos oficiales.  

El manejo incorrecto de esta información puede provocar problemas como:

- Falsificación de documentos
- Alteración de información
- Suplantación de identidad
- Acceso no autorizado a datos personales

Este proyecto propone el diseño de un sistema criptográfico que permita:

- Proteger datos personales
- Firmar digitalmente documentos
- Verificar la autenticidad de documentos emitidos
- Gestionar identidades digitales
- Garantizar integridad, autenticidad y no repudio

El objetivo es mejorar la seguridad digital de organizaciones sociales sin aumentar la complejidad de sus procesos operativos.
